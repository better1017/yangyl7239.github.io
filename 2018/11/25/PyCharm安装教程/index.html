PyCharm安装教程

发表于 2018-11-25 分类于 安装教程 阅读次数：

下载

  • 官网下载安装包，这里选择professional版本。

安装

开始安装，选择next

选择安装路径，继续next

选择安装选项，继续next

进入这一页面，默认选择，点击install

安装中，等待就好了

安装完毕

因为在前边安装选项中的时候我们选择了Add launchers dir to the path，所以需要重启电脑，这里有两个选项：

  • 现在立即重启

  • 稍后手动重启

选择立即重启，因为我们要立刻建一个python项目尝试一下。

用PyCharm新建第一个python项目

这里我实际上是已经安装了anaconda，所以电脑里已经有了python环境，而且也已经配置了anaconda的环境变量（D:\Program Files\anconda和D:\Program Files\anconda\Scripts配置到Path）

软件初始配置

  • 配置导入，选择不导入

  • 主题选择，选择Darcula的暗黑系主题

  • 开始使用

新建一个项目

选择路径

  • Location：项目路径

  • Base interpreter：python.exe的路径，这里我是安装了anaconda，所以已经有了python环境，而且我还配置了环境变量，所以PyCharm可以自动找到python.exe的路径。点击creat。

新建Python文件

项目名处右键 -> new -> python file -> test01 -> OK

更改编辑器区域字体大小
