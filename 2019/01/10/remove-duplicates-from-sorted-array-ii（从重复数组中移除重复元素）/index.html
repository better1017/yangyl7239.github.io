发表于 2019-01-10 分类于 算法 阅读次数：

类似问题：remove duplicates from sorted list（移除有序链表中的重复元素）

题目描述

Follow up for “Remove Duplicates”:
What if duplicates are allowed at most twice?
For example,
Given sorted array A =[1,1,1,2,2,3],
Your function should return length =5, and A is now[1,1,2,2,3].

题目大意

从重复数组中移除重复元素；
允许最多重复两次。

方法一

思路

常规版本。
设置index索引值，索引数组序号，然后对数组进行遍历；
当有前中后三个连续重复元素时，就执行continue；
否则就把当前值赋值给index下标，并且把index加一。
